Project Webdesign

Welkom op mijn website. Deze website heeft als doel, jou meer inzicht te geven over mijn traject op het school, als ook de ervaringen die ik al heb gehad in het werkveld.

Je zal op deze pagina kunnen navigeren naar mijn hobbies en interesses, maar ook naar de webpagina waar je kan zien welke competenties we hebben geleerd op het school of aan het leren zijn. Hopelijk vind je de website interessant genoeg.

Wie ben ik?... Kom meer over mij te weten

Ik ben Jeroen Piussi een student aan Thomas More Kempen in Geel. Voor het vak Webdesign kregen we een project waarin we een portfolio website moesten bouwen. Hiervoor heb ik deze website gemaakt. Ik heb een passie voor technologie, hardware en webdesign. Dit hoop ik ook te kunnen uitdragen in mijn toekomstige werk.
